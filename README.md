# 🩺 Doctor Appointment Web Application

A modern and user-friendly Doctor Appointment Management System that allows patients to easily book appointments with doctors online. The application provides a seamless experience for scheduling appointments, managing patient information, and organizing doctor availability.

## 🚀 Features

- 👨‍⚕️ Doctor registration and login
- 👤 Patient registration and login
- 📅 Book doctor appointments
- 📝 View and manage appointment history
- 🔍 Browse available doctors
- ⏰ Appointment scheduling
- 📱 Responsive and user-friendly interface
- 🔐 Secure authentication
- 💻 Modern and clean UI

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- HTML5
- CSS3
- JavaScript (ES6+)

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/doctor-appointment.git
```

### Install dependencies

#### Frontend

```bash
cd Frontend
npm install
npm run dev
```

#### Backend

```bash
cd Backend
npm install
npm start
```
